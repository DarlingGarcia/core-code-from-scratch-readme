## String: length


## String: toUpperCase() and toLowerCase()

```
function toCase(name){
  return name.toLowerCase() + '-' + name.toUpperCase()
}
```

## String: charAt()

```
function shortcut(name1, name2){
  return name1.charAt(0) + name2.charAt(0)
}
```
